# Proyecto Python - Práctica Git

Proyecto de práctica para aprender Git y GitHub

## Funcionalidades:
- Calculadora básica
- Utilidades matemáticas

## Instalación
```bash
git clone https://github.com/Oscararagon27/practica-python.git
cd practica-python
python3 -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate
```

## Uso
```bash
python src/main.py
```
