# Proyecto Python - Práctica Git

Proyecto de práctica para aprender Git y GitHub

## Funcionalidades:
- Calculadora básica
- Utilidades matemáticas

## Instalación
```bash
git clone [url-del-repositorio]
cd practica-python
python3 -m venv venv
source venv/bin/activate  # En Windows: venv\Scripts\activate
```

## Uso
```bash
python src/main.py
```
